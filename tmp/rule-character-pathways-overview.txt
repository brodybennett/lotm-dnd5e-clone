Character Pathways In this system, a character progresses through a single Pathway from Sequence 9 to Sequence 0 . Sequence advancement replaces normal class leveling and multiclass progression for standard play. Core Pathway Principles Sequence progression is both quantitative and qualitative: higher tiers unlock different categories of influence, not only larger numbers. Power is not damage only. Utility, control, information, ritual leverage, survivability, and narrative authority are all valid budget spend. Pathways can be asymmetric in role while still remaining within the same sequence power band. Corruption pressure is part of balance at higher sequence authority. Pathway Creation Workflow Write a one-sentence sequence fantasy: what this sequence enables that the previous sequence could not. Set the budget target (total package or gain package). Choose a power vector (offense, control, utility, survivability, information, ritual leverage, authority). Draft 3-6 features (signature, support, passive, optional ritual). Score each feature by impact, reliability, scope, duration, and limits. Validate domain coverage and intentional weaknesses. Run abuse checks and add GM adjudication notes where needed.
